## Installation

```bash
npm install -g @type-zen/cli
```

## Usage

```bash
# Directory structure
- a.tzen
- b.tzen
```

Compile only `a.tzen` :

```bash
tzc a.tzen
```

Compile all files in the current directory:

```bash
tzc ./*.tzen
```
